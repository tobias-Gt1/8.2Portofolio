# Tobias — Web Developer Portfolio

Een bold & creatief, volledig responsive portfolio gebouwd met React, Vite, TypeScript en Tailwind CSS.

## Features

- Tweetalig (NL/EN) met taal-toggle in navbar
- Secties: Hero, About, Skills, Projects, Experience, Contact
- Scroll animaties via Intersection Observer
- Mobile-first responsive design
- Data-driven content via `src/data/content.ts`

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in je browser.

## Content aanpassen

Alle persoonlijke content staat in [`src/data/content.ts`](src/data/content.ts). Pas daar je naam, bio, skills, projecten en ervaring aan.

UI-teksten (knoppen, labels) staan in [`src/i18n/translations.ts`](src/i18n/translations.ts).

## Build

```bash
npm run build
npm run preview
```
